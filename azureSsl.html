Certificado SSL

¿Qué es SSL?

Las siglas significan “Secure Sockets Layer”,es un certificado digital el cual ayuda a que el sitio web sea más segura y confiable la trasmisión de información entre un servidor web y otro punto. Se fundamenta en un ciclo de encriptación y desencriptación de los datos.


Por ejemplo, en caso de que se llene un formulario en la web que no tenga un certificado SSL los datos se transfieren a su destino sin una especie de seguridad o encriptación, lo cual, es un riesgo, ya que cualquier persona puede ver lo que se envía a internet.


Un certificado SSL es importante ya que SEO GOOGLE penaliza a aquellos sitios web que no tengan certificado SSL potente, ya que no se debe de contratar cualquier certificado e incluso hay empresas que certifican, pero con baja calidad y Google las considera inseguras. Sin embargo, se puede obtener un certificado SSL con solo contratar un servicio de hosting que ya lo tenga integrado.

Tipos de SSL

Validación ampliada

Es el nivel más alto de seguridad con un proceso exhaustivo para verificar la máxima seguridad de un sitio web.

Validación de organización

Nivel intermedio, menos exhaustivo, pero una muy buena solución para las páginas web.

Validación de dominio

Nivel esencial, es fácil de obtener y es bastante económico por regla general.

Azure APP Service es un servicio basado en HHTP para hospedar aplicaciones web, API REST y backends móviles, ofrece un hospedaje escalable y con parcheo automático.

Opción 1

Crear un certificado administrado de App Service gratuito es fácil de usar y solo se necesita proteger el dominio personalizado en App Service. .

Opción 2

Aplicar un certificado privado.

Para ello se necesita lo siguiente:

• Exportar el Certificado como un archivo PFX protegido con contraseña , cifrado con triple DES.

• Contener una clave privada de al menos 2048 bits de longitud

Para proteger un dominio personalizado en un enlace TLS, el certificado tiene requisitos adicionales: Contiene un uso de clave extendido para la autenticación del servidor (OID = 1.3.6.1.5.5.7.3.1) Firmado por una autoridad certificadora de confianza
